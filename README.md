# Campus Connect

A comprehensive campus resource booking and management system built with PHP MVC architecture. Streamlines the process of booking campus facilities, managing endorsements, tracking resource utilization, and campus announcements.

## Features

### Student Features
- **Browse Resources** - Explore available campus facilities (study rooms, labs, projectors, sports facilities, auditoriums)
- **Book Resources** - Submit booking requests with date, time range, and purpose
- **Track Bookings** - Monitor booking status (Pending, Approved, Rejected, Cancelled, Completed)
- **Booking History** - View complete booking history with detailed statistics
- **Search & Filter** - Find specific resources by type, location, or availability
- **Campus Announcements** - Stay updated with clubs, conferences, and research opportunities
- **Check-In/Check-Out** - Track attendance with automated timestamps
- **Resource Reviews** - Rate and review resources after use

### Faculty Features
- **Endorse Requests** - Review and approve/reject student booking requests
- **Bulk Booking Requests** - Request multiple resources for events
- **Dashboard Analytics** - View endorsement statistics and trends
- **Monthly Reports** - Track endorsement activity over time
- **Recent Activity** - Monitor latest booking requests
- **Post Announcements** - Create club, conference, and research opportunities
- **Attendance Management** - Mark student attendance for events

### Admin Features
- **Central Dashboard** - Manage all bookings from a single interface
- **User Management** - Oversee students and faculty accounts with status control
- **Resource Management** - Add, edit, and remove campus resources
- **Approval Workflow** - Final approval authority for all bookings
- **System Analytics** - Comprehensive statistics and reports
- **Real-time Monitoring** - Track system-wide booking activity
- **Ticket Management** - Handle maintenance and issue reports
- **Admin Logs** - Complete audit trail of all administrative actions
- **Notification System** - Send alerts and reminders to users

## Tech Stack

- **Backend**: PHP 8.0+
- **Architecture**: MVC (Model-View-Controller)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Bootstrap 5.3, Custom CSS with Gradients
- **Icons**: Bootstrap Icons
- **Server**: Apache (XAMPP)
- **Version Control**: Git

## Prerequisites

- XAMPP (or similar PHP development environment)
- PHP 8.0 or higher
- MySQL 5.7+ or MariaDB 10.4+
- Modern web browser (Chrome, Firefox, Edge)
